# ToDo App – Task Manager

A simple and responsive ToDo application to manage daily tasks efficiently. Users can add, update, delete, and track tasks in real-time using JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Real-time updates
- Clean and responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ⚙️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

## 📌 Future Improvements
- Add local storage support
- Add task categories
